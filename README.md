## Objetivo

Implementar algoritmos clássicos de busca para resolver problemas de IA.

## Problemas

### Régua-Puzzle

O Régua-Puzzle é um problema clássico de IA que consiste em encontrar os passos de um estado inicial até um estado final. Cada estado é formado por uma sequência com a mesma quantidade de peças azuis (`A`) e brancas (`B`), além de um espaço vazio (`-`).

Neste projeto, o objetivo é encontrar um estado final em que todas as peças brancas estejam à esquerda das peças azuis.

## Algoritmos

### Métodos Não Informados de Busca

São métodos que não usam heurística auxiliar para orientar a busca.

1. Backtracking
2. Breadth-First Search
3. Depth-First Search
4. Ordered Search

### Métodos Informados de Busca

São métodos que usam heurística auxiliar para orientar a busca.

1. Gulosa
2. A*
3. IDA*

## Resultado Esperado

É esperado o resultado para a execução de cada algoritmo:

1. Caminho da solução
2. Custo da solução
3. Nós expandidos e visitados
4. Fator de ramificação médio
5. Tempo de execução
