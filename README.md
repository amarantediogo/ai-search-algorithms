## Objetivo

O objetivo deste projeto é implementar os ALGORITMOS clássicos de IA para resolver os PROBLEMAS.

## Problemas

### Régua-Puzzle

O régua puzzle é um problema clássico de IA que consiste em encontrar os passos (caminho) para dado um estado inicial formado por um sequências de peças azuis (A), brancas (B) e um espaço vazio (-), tal que o número de peças azuis e o de peças brancas são iguais. O objetivo é encontrar um estado final onde todas as peças azuis estão a esquerda das brancas.

## Algoritmos

### Métodos Não Informados de Busca

São métodos onde não existe uma heurísticas auxiliar para a busca.

1. Backtracking
2. Breadth-First Search
3. Depth-First Search
4. Oredered Search

### Métodos Informados de Busca

São métodos onde existe uma heurísticas auxiliar para a busca.

1. Gulosa
2. A-star
3. IDA-star
